무료 PDF 편집과 무료 PDF 압축, 설치 없이 한 곳에서

CgangPDF는 회원가입 없이 쓰는 무료 PDF 편집·압축 사이트입니다. PDF 페이지 삭제, 순서 변경, PDF 용량 줄이기를 프로그램 설치 없이 브라우저에서 바로 처리하세요.

📄 무료 PDF 편집썸네일을 드래그해서 페이지 순서를 바꾸고, 필요 없는 페이지는 ✕ 버튼으로 삭제합니다. 확대 보기에서 펜·형광펜으로 필기하거나 페이지를 자를 수도 있습니다. 원본 화질 그대로(무손실) 저장됩니다.
🗜️ 무료 PDF 압축저화질·중화질·고화질 3단계 중에서 선택해 PDF 용량을 줄입니다. 스캔본이나 이미지 위주 PDF에서 효과가 큽니다.
🔄 PDF ↔ 이미지 변환JPG·PNG·GIF 등 여러 장을 순서대로 묶어 하나의 PDF로 만들거나, 반대로 PDF의 각 페이지를 JPG 이미지로 추출합니다.
🔒 완전한 프라이버시파일이 서버로 전송되지 않습니다. 모든 처리가 내 브라우저 안에서만 이루어지고, 탭을 닫으면 흔적 없이 사라집니다.

PDF 편집·압축 사용 방법

  1. 작업 선택 — 첫 화면에서 PDF 편집·압축 또는 파일 형변환을 선택합니다.
  2. 파일 열기 — 점선 상자에 파일을 끌어다 놓거나, 클릭해서 선택합니다. (최대 100MB)
  3. 페이지 확인 — 모든 페이지가 썸네일로 나타납니다. 썸네일을 클릭하면 전체화면으로 크게 볼 수 있고, 좌우 화살표(← →)로 다음·이전 페이지를 넘겨볼 수 있습니다.
  4. 편집 — 썸네일을 드래그해서 순서를 바꾸고, ✕ 버튼으로 필요 없는 페이지를 삭제 표시합니다. 실수로 삭제했다면 해당 페이지의 버튼을 한 번 더 누르면 복원됩니다.
  5. 저장 — 화질을 유지하려면 적용 & 다운로드(무손실), 파일 크기를 줄이려면 압축에서 원하는 단계를 선택하세요. 편집한 순서와 삭제가 그대로 반영됩니다.

자주 묻는 질문

내 PDF 파일이 서버에 업로드되나요?

아니요. CgangPDF에는 파일을 받는 서버 자체가 없습니다. PDF를 열고, 편집하고, 압축하는 모든 과정이 지금 보고 계신 브라우저 안에서만 실행됩니다. 인터넷 연결이 끊긴 상태에서도 동작할 정도로 완전한 로컬 처리 방식입니다.

"적용 & 다운로드"와 "압축"의 차이가 뭔가요?

적용 & 다운로드는 페이지를 그대로 복사하는 무손실 저장이라 화질 저하가 전혀 없습니다. 압축은 각 페이지를 이미지로 다시 그려서 저장하기 때문에 파일이 작아지는 대신 화질이 조금 낮아지고, 텍스트 선택·검색이 안 되게 됩니다. 순서 변경·삭제만 필요하면 "적용 & 다운로드"를 쓰세요.

압축했더니 파일이 오히려 커졌어요. 왜 그런가요?

글자가 텍스트(폰트 정보)로 저장된 일반 문서 PDF는 원래 용량이 매우 작습니다. 이런 파일을 이미지 방식으로 압축하면 페이지 전체가 사진으로 바뀌면서 용량이 오히려 늘어날 수 있습니다. CgangPDF는 이런 경우를 감지해서 저장 전에 알려드립니다. 압축은 스캔본, 촬영본, 캡처본처럼 이미 이미지로 이루어진 PDF에서 효과가 큽니다.

어떤 압축 단계를 선택해야 하나요?

화면으로만 볼 파일이라면 중화질(150dpi)이 크기와 화질의 균형이 가장 좋습니다. 메일 첨부 등 용량이 최우선이면 저화질(72dpi), 인쇄할 가능성이 있다면 고화질(300dpi)을 권장합니다.

100MB보다 큰 파일은 어떻게 하나요?

브라우저 메모리 한계 때문에 안정적인 처리를 위해 100MB로 제한하고 있습니다. 더 큰 파일은 먼저 앞뒤로 나누어 처리하거나, 데스크톱용 도구(Ghostscript 등)를 사용하는 것을 권장합니다.

JPG 사진을 PDF로 바꿀 수 있나요?

네. 첫 화면에서 JPG·JPEG·PNG·WebP·GIF·BMP·AVIF·SVG 이미지를 여러 장 선택하거나 끌어다 놓으면 썸네일이 나타나고, 드래그로 순서를 정한 뒤 변환하기를 누르고 최종 형식(PDF·JPG·PNG·WebP)을 선택하면 변환됩니다. 반대로 PDF를 열고 이미지로 추출을 누르면 각 페이지가 선택한 이미지 형식으로 변환되어 ZIP 파일로 저장됩니다.

PDF를 워드(DOCX)나 한글(HWP)로 바꿀 수 있나요?

HWP는 한글과컴퓨터의 비공개 포맷이라 웹 기술로는 변환이 불가능하고, 워드 변환도 레이아웃을 온전히 살리기 어렵습니다. 대신 CgangPDF는 TXT 텍스트 추출을 제공합니다 — 파일 형변환에서 PDF를 열고 TXT를 선택하면 글자만 추출되어, 워드나 한글에 붙여넣어 편집할 수 있습니다. 단, 스캔본처럼 이미지로만 된 PDF에는 글자 정보가 없어 추출되지 않습니다.

무료인가요? 회원가입이 필요한가요?

완전 무료이며 회원가입도 필요 없습니다. 파일 개수나 사용 횟수 제한도 없습니다.

지원하는 브라우저는?

Chrome, Edge, Safari, Firefox 등 최신 브라우저를 모두 지원합니다. 데스크톱 환경에서 드래그 정렬이 가장 편리하며, 모바일에서는 보기와 삭제 기능 위주로 사용하는 것을 권장합니다.

PDF 용량 줄이기 팁 — 압축 전에 알아두면 좋은 것

  • 필요 없는 페이지부터 삭제하세요. 압축보다 페이지 삭제가 용량 감소 효과가 확실하고 화질 손해도 없습니다. 표지·백지·광고 페이지만 지워도 크게 줄어드는 경우가 많습니다.
  • 스캔본은 압축 효과가 큽니다. 스캐너나 카메라로 만든 PDF는 페이지 전체가 고해상도 이미지라서, 중화질 압축만으로도 50~80% 줄어드는 것이 일반적입니다.
  • 텍스트 PDF는 압축하지 마세요. 워드·한글에서 내보낸 문서형 PDF는 이미 충분히 작습니다. 압축하면 커질 수 있고 글자 검색 기능을 잃습니다.
  • 중요한 파일은 원본을 보관하세요. 압축은 되돌릴 수 없으므로, 원본을 지우지 말고 압축본과 함께 보관하는 것이 안전합니다.
