CgangPDF는 어떤 서비스인가요?

CgangPDF는 PDF의 페이지 순서 변경, 페이지 삭제, 파일 압축을 무료로 제공하는 웹 도구입니다. 설치할 프로그램도, 회원가입도 없습니다. 브라우저에서 페이지를 열고 PDF를 끌어다 놓으면 바로 사용할 수 있습니다.

왜 만들었나요?

PDF에서 페이지 몇 장을 지우거나 순서를 바꾸는 아주 단순한 작업조차, 기존 온라인 도구들은 파일을 자신들의 서버에 업로드하도록 요구합니다. 계약서, 신분증 사본, 업무 문서처럼 민감한 파일을 어딘지 모르는 서버에 올리는 것이 늘 찜찜했습니다.

CgangPDF는 그 찜찜함을 없애기 위해 만들어졌습니다. 파일을 받을 서버를 아예 두지 않는 것이 이 서비스의 출발점입니다.

어떻게 서버 없이 동작하나요?

최신 브라우저는 그 자체로 훌륭한 실행 환경입니다. CgangPDF는 검증된 오픈소스 라이브러리 두 가지를 브라우저 안에서 직접 실행합니다.

  • pdf.js (Mozilla) — PDF를 읽고 페이지를 화면에 그리는 렌더링 엔진. Firefox 브라우저에 내장된 그 엔진입니다.
  • pdf-lib — 페이지 복사·재배열 등 PDF 문서를 조립하는 편집 엔진입니다.

파일을 열면 브라우저 메모리 안에서만 읽히고, 편집된 결과물은 곧바로 내 컴퓨터의 다운로드 폴더로 저장됩니다. 네트워크로 파일이 전송되는 구간이 존재하지 않으며, 탭을 닫으면 메모리에서 완전히 사라집니다.

기능 요약

  • 페이지 썸네일 보기, 전체화면 확대 보기(화살표로 페이지 넘기기)
  • 드래그로 페이지 순서 변경, 클릭 한 번으로 페이지 삭제
  • 무손실 저장(적용 & 다운로드) — 화질 저하 없음
  • 3단계 압축(저·중·고화질) — 스캔본·이미지형 PDF에 효과적
  • 압축 결과가 원본보다 커지면 저장 전에 알려주는 안전장치
